# PHP-FPM & FFMPEG Docker Image
This image is a customization of [PHP-FPM 5.3, 5.4, 5.5, 5.6, 7.0, 7.1, 7.2, 7.3, 7.4 Docker image](https://github.com/Rp70/docker-php) & the latest [static build](https://www.johnvansickle.com/ffmpeg/) of [FFmpeg](https://www.ffmpeg.org/). This image is used to develop video & audio web application.

All PRs and feedback are welcome!

# Helpful Links
* [PHP-FPM 5.3, 5.4, 5.5, 5.6, 7.0, 7.1, 7.2, 7.3, 7.4 Docker image](https://github.com/Rp70/docker-php)
* [FFMPEG static builds](https://www.johnvansickle.com/ffmpeg/): the latest versions of FFmpeg for Linux kernels 3.2.0 and up.
* [FFmpeg](https://www.ffmpeg.org/): A complete, cross-platform solution to record, convert and stream audio and video.
* [PHP-FFMpeg](https://github.com/PHP-FFMpeg/PHP-FFMpeg): An Object-Oriented library to convert video/audio files with FFmpeg / AVConv.
